# reload-price-farpost
Скрипт на Python двигающий курсор, обновляя цены товаров на сайте farpost.ru

На сайте есть товары, которые создаются вручную и по прайс-листу. Вручную созданные товары соответственно поднимаются в рейтинге выше, но цены надо изменять руками. По идее надо было написать парсер, но времени было мало, а на сайте всюду капчи, я не стал заморачиваться, сделал говно.
